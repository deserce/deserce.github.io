通达信榜单设置（一）—— 实现某app付费功能！

前言

这里先不探讨榜单的使用方法，只记录在通达信设置榜单的方法。
这个系列文章主要完成三个榜单的设置：

  • 北向资金持股
  • 公募基金持股
  • RPS（股价相对强弱）

至于为什么做这三个榜单呢？是因为我发现在某些app里面这些榜单是会员专享的，当然土豪想买的话随意。所以我秉持着白嫖的思想，通过在网上查询资料寻找在通达信中实现类似功能的方法。有心人天不负，终于被我完成了。

差别还是蛮大的，怀着一颗探求真理的心，我挑了一个股票来自己算了一下他的近10日涨幅。

我选择的是创业黑马，写下这篇文章的时间是2021年9月3日，当天的收盘价是38.44，然后前10个交易日是2021年8月20日，当天的收盘价是18.39，然后就是计算近10日涨幅。

Code

翻译一下就是：输出近10日涨幅:(收盘价-10日前的收盘价)/10日前的收盘价

所以 （38.44-18.39）/18.39*100%=109.026%

而某app算出来是75.4%，难道他是将非交易日也算进去了？那我们来算一下看看？前10天那就是8月28日而那一天是礼拜六，所以是横盘，我们用前一个交易日8月27的收盘价来算一下，当日收盘价为19.80。

所以 （38.44-19.80）/19.80*100%=94.14%

好吧还是对不上，如果哪位大佬知道原因，望大佬赐教。

我的结论就是还是以通达信为准，貌似准确率会比较高一些。
